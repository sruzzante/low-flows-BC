The "daily_*.csv" files are discharge time series from 463 unregulated (natural) streams in BC.
Download from:
https://wateroffice.ec.gc.ca/search/historical_results_e.html?search_type=province&province=BC&start_year=1850&end_year=2024&minimum_years=20&regulation=N&gross_drainage_operator=%3E&gross_drainage_area=&effective_drainage_operator=%3E&effective_drainage_area=

The file "stationList.csv" contains metadata from the search results (copy-pasted).

daily_*.csv files are excluded from repository due to file size
